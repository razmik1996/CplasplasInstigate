Introduction

Solution for exercises 15 to 28.

  1. Minimum and maximum Write a program to input 10 numbers and find the minimal and maximal numbers in the sequence. Print the results.
  2. Occurence of numbers Write a program which will allow user to input numbers in range 1-10 (as much as the user would like to, till -1 is entered). Calculate and print how many times each of the numbers have been entered.
  3. Decimal to binary Write a program to input a number and convert it to binary (e.g in 4 bits) e.g Input 5 , Output 0101 Enhance the program to support also negative numbers (calculate 2's complement)
  4. Binary to decimal Write a program to input a binary number (a sequence of 0-1s) and convert it to decimal. Print the results.
  5. Print the sum of all numbers in the given sequence of N whole numbers.
  6. Input a sequence of N numbers and sort the sequence in ascending/descending order.
